# Bug ou Verdade
Este projeto foi desenvolvido para uma materia extensionista da faculdade, com o intuito de integrar o estudo com a pratica.
É um jogo simples no estilo de perguntados, porém com uma dinamica de pontuação baseada no tempo de resposta, a ideia foi aplicar o jogo em uma escola técnica, sendo possivel mensurar o nivel de conhecimento com perguntas de programação simples antes de chegar no nivel de graduação.


## Funcionalidades
- Perguntas com validações de respostas
- Pontuação baseada em tempo de resposta (mais rapido = mais pontos)
- Tempo maximo para responder perguntas
- Mostra a pontuação e respostas corretas após terminar o jogo


## Tecnologias Utilizadas
* [LUA](https://www.lua.org/)
* [LOVE2D](https://love2d.org/)
 
### Pré-requisitos

Tenha instalado:

* Linguagem Lua
* Biblioteca LOVE2D

### Instalação

- Clone o repositório:
    ```bash
    git clone https://github.com/IgorPirola/Bug_ou_Verdade.git
    cd Bug_ou_Verdade
    ```

### Execução

- Faça uma variavel de ambiente ou execute diretamente com o arquivo do love:
   ```bash
   "C:\Program Files\LOVE\love.exe" "C:\caminho\Bug_ou_Verdade"
   ```
   (Substitua o caminho conforme a instalação do LOVE e onde você clonou o projeto.)
